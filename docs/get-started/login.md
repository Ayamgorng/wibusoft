# Login

### 2. Klik Login

### 3. Masukan Email dan Password yang sudah kamu daftarkan

- Centang `Remember me` agar kamu tidak perlu login ulang terus menerus
- Pastikan jaringan kamu stabil dan centang `Saya bukan robot`
- Klik Login
