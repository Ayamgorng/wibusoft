## Command

### #buylimit

Penggunaan dapat dilakukan dengan cara mengirim `#buylimit jumlah_limit`.

Untuk membeli limit, kamu harus menukarkan `100 balance dengan 1 limit`.

Contoh penggunaan:

- #buylimit 3.
