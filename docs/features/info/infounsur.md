## Command

### #infounsur

Penggunaan dapat dilakukan dengan cara mengirim `#infounsur query`.

query disini dapat diisi dengan:

- Nomor atom
- Simbol Unsur
- Nama Unsur (Inggris)

Alias:

- #unsur
