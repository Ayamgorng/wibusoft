## Command

### #groupinfo

Penggunaan dapat dilakukan dengan cara mengirim `#groupinfo`.

Bot akan memberikan info tentang group diantaranya:

- `ID_GROUP`
- `NAMA_GROUP`
- `JUMLAH_ANGGOTA_GROUP`
- `PEMBUAT_GROUP`
- `KAPAN_GROUP_DIBUAT`
- `GROUP_SETTING` seperti antidelete, antilink, dll.
- `DESKRIPSI_GROUP`

Alias:

- #grupinfo
- #infogroup
- #infogrup
