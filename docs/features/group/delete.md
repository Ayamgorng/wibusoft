# Delete Message

Berfungsi untuk mengehapus pesan di group.

|                       Info                        |      |
| :-----------------------------------------------: | :--: |
| <div class="label license nitro">Only Nitro</div> |  ❌  |
|                Menggunakan Prefix                 |  ✅  |
|                 Menggunakan limit                 |  ❌  |
|                    Hanya Owner                    |  ❌  |
|                   Hanya Premium                   |  ❌  |
|                    Hanya Group                    |  ✅  |
|                 Hanya Admin Group                 |  ✅  |
|                Hanya Private Chat                 |  ❌  |
|                       Limit                       |  1   |
|                     Cooldown                      | 5000 |
|                       Level                       |  0   |

## Command

### #delete

Reply pesan yang ingin di hapus dengan caption `#delete`.

Alias :

- #d
- #del

## Note

Kamu dapat menghapus pesan dari orang lain, jika bot merupakan admin grup.
